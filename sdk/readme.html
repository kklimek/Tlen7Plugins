Przykładowy plugin

Otwieramy konsole Visual Studio, wchodzimy do katalogu z przykładowym pluginem, następnie wydajemy komendę "cmake .". Po utworzeniu pliku solucji otwieramy ją w Visual Studio i kompilujemy. Utworzony plugin wrzucamy do podkatalogu plugins w katalogu Tlenu. Plugin trzeba jeszcze włączyć w ustawieniach programu.
