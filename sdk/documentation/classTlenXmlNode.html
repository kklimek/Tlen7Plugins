TlenXmlNode TlenXmlNode::getChild ( const QString &  childPath  )  const

Zwraca dziecko o podanej sciezce.

Elementy sciezki rozdzielamy znakiem /
Np. dla XML-a: 

 <a>
        <b>
                <c/>
        </b>
 </a>
Wywolanie getChild("b/c"), na wezle 'a' zwroci wezel 'c'
Parametry:
childPath sciezka dziecka ktore chcemy wyciagnac
Zwraca:
wezel XML, pusty jesli nie znaleziono dziecka
